### 补充条款
#### 关于商用
在遵顼以下条款的前提下,洛书及其源代码被允许用作商业用途
1. 从本仓库授权开发者手中获得商业使用授权
2. 在您的商用版本中标注商用授权

#### 关于洛书支持库
一般情况下,洛书拓展库不被视为洛书的一部分,无论是否由本仓库成员开发,不用遵循洛书所遵循的开源协议<br>
但如果您希望将其开源<br>
您可以选择将其提交到本仓库，并注明所用开源协议。
